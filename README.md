# jkRFExtensionSettings

Helper module to manage RoboFont extension settings.

### Example

```
def test():
    my_settings = SettingsWindow("de.kutilek.test", "My Settings")
    my_settings.add("mySlider", 0.0, "My Slider")
    my_settings.add("myCheckbox", True)
    my_settings.show()
```
